pub fn get_inner_array_types<'a>( &'a self, types: &mut Vec<&'a DataTypeInformation>, index: &'a Index )

Recursively retrieves all type names for nested arrays.

This is needed because a nested array such as foo : ARRAY[1..5] OF ARRAY[5..10] OF DINT provides range information for [1..5] and [5..10] in two different types stored in the index.

pub fn get_array_length(&self, index: &Index) -> Option<usize>

Returns the array length if DataTypeInformation is of variant DataTypeInformation::Array and None otherwise.

For example calling this function on ARRAY[1..5] OF DINT, ARRAY[1..2, 1..5] OF DINT and ARRAY[1..3] OF ARRAY[1..5] yields 5, 10 and 15 respectively.
