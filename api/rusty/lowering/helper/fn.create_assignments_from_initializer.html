Expand description

Takes some expression such as foo : FooStruct := (bar := (baz := (qux := ADR(val)), baz2 := (qux := ADR(val)))); and returns assignments of form [foo.bar.baz.qux := ADR(val), foo.bar.baz2.qux := ADR(val)].

For REF() calls on the RHS (e.g., foo := REF(bar)), this generates REF= assignments instead, so foo REF= bar which properly sets up reference semantics.
