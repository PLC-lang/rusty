unsafe fn encode(self, dest: &mut *mut T)

Encodes UTF-8 or UTF-16 character iterator. Its raw codepoints are written into given destination buffer address.

§Safety

Works on raw pointers, inherently unsafe. Does not ensure that the buffer at the given address large enough. It will continue to write unchecked until all characters have been processed and can therefore result in UB.
