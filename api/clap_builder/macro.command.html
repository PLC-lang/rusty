clap_builder

Macro command

source 
macro_rules! command {
    () => { ... };
    ($name:expr) => { ... };
}
Expand description

Allows you to build the Command instance from your Cargo.toml at compile time.

NOTE: Changing the values in your Cargo.toml does not trigger a re-build automatically, and therefore won’t change the generated output until you recompile.

In some cases you can “trick” the compiler into triggering a rebuild when your Cargo.toml is changed by including this in your src/main.rs file include_str!("../Cargo.toml");
