fn use_init_array(&mut self, use_init_array: bool)

Set whether to use .init_array and .fini_array sections for global constructors and destructors instead of the legacy .ctors and .dtors sections.

By default, LLVM uses .ctors and .dtors sections. This method allows changing that behavior.
