Set the last modification time for a file on the filesystem.

This function will set the mtime metadata field for a file on the local filesystem, returning any error encountered.

§Platform support

Where supported this will attempt to issue just one syscall to update only the mtime, but where not supported this may issue one syscall to learn the existing atime so only the mtime can be configured.
