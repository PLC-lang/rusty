Struct FixedMersenne

pub struct FixedMersenne<const P: u8, const K: umax>();
Expand description

A modular reducer for (pseudo) Mersenne numbers 2^P - K as modulus. It supports P up to 127 and K < 2^(P-1)

The P is limited to 127 so that itโs not necessary to check overflow. This limit wonโt be a problem for any Mersenne primes within the range of umax (i.e. u128).
