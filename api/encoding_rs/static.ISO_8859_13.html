The ISO-8859-13 encoding.

This is the Baltic part of the ISO/IEC 8859 encoding family. This encoding is also known as Latin 7.

Index visualization, Visualization of BMP coverage

This encoding matches the Windows code page 28603, except Windows decodes unassigned code points to the Private Use Area of Unicode.

This will change from static to const if Rust changes to make the referent of pub const FOO: &'static Encoding unique cross-crate, so don’t take the address of this static.
