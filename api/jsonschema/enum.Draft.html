pub fn from_schema_uri(uri: &str) -> Draft

Detect a draft from a $schema URI string.

Accepts the URI with or without a trailing #. Returns Draft::Unknown for custom/unknown meta-schema URIs.

Source

pub fn detect(self, contents: &Value) -> Draft

Detect what specification could be applied to the given contents.

Inspects the $schema field and returns the matching draft. If no $schema field is present, returns self unchanged — the caller’s current draft is preserved.

Returns Draft::Unknown for custom/unknown $schema values. Validation of custom meta-schemas happens during registry building.
