pub fn instance(&self) -> Option<&str>

The reference a block’s outputs are read through: its declared instance (function block), falling back to the owner POU (a program singleton, or an action’s owner — outputs live on the instance, not the action).

pub fn return_pin(&self) -> Option<&Pin>

The output pin carrying a function’s return value: the pin named after the callee. Only a non-void function exposes one.

Source

pub fn is_function(&self) -> bool

A stateless function call: no caller instance and a return pin present. Its outputs don’t persist, so they are read through generated temporaries rather than instance.member.
