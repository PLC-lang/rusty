plc_cfc

Module model 

Source
Expand description

Deserialization model for CFC (.cfc) files: a PLCopen-style XML document describing a single POU whose body is an FBD/CFC network.

quick-xml’s serde support strips namespace prefixes and matches on the local name, so the renames use bare names (Function, type) rather than their ppx: / xsi: prefixed forms, and attributes take a leading @.
