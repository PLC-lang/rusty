Source

fn is_empty(&self) -> bool

Returns whether or not this generated header is empty

This must return true if the generation process has not yet occured, or was aborted in a valid case.

Source

fn get_contents(&self) -> &str

Returns the contents of the header file

Source

fn prepare_template_data(&mut self, compilation_unit: &CompilationUnit)

Prepares the template data for this header given a compilation unit

The outcome of this method must be a populated TemplateData on the generated header that contains all of the data necessary to run the templating engine.

Source

fn generate_headers(&mut self) -> Result<(), Diagnostic>

Runs the templating engine and generates the header file contents

The outcome of this method must be a populated “contents” (accessible via the “get_contents” method) on the generated header.
