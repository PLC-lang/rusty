Encode text to prevent special characters functioning and write it to a mutable Vec<u8> reference and return the encoded data slice.

The following characters are escaped:

  • & => &amp;
  • < => &lt;
  • > => &gt;
  • " => &quot;
  • ' => &#x27;
  • / => &#x2F;
