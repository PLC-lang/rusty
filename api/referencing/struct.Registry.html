Struct Registry

pub struct Registry<'a> { /* private fields */ }
Expand description

A registry of JSON Schema resources, each identified by their canonical URIs.

Registry is a prepared registry: add resources with Registry::new and RegistryBuilder::add, then call RegistryBuilder::prepare to build the reusable registry. To resolve $ref references directly, create a Resolver from the prepared registry:

use referencing::Registry;

let schema = serde_json::json!({
    "$schema": "https://json-schema.org/draft/2020-12/schema",
    "$id": "https://example.com/root",
    "$defs": { "item": { "type": "string" } },
    "items": { "$ref": "#/$defs/item" }
});

let registry = Registry::new()
    .add("https://example.com/root", schema)?
    .prepare()?;

let resolver = registry.resolver(referencing::uri::from_str("https://example.com/root")?);
