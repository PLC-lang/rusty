# Invalid Enum Element Expression

Enum elements must be either simple identifiers or assignments (e.g., `ElementName := Value`).

## Example

```st
// Incorrect
TYPE State : (
    Idle,
    Running(), // Error: a call-statement cannot be an enum variant
    Stopped
);
END_TYPE

// Correct
TYPE State : (
    Idle,
    Running := 3,
    Stopped
);
END_TYPE
```

## Explanation

Enum element definitions have a restricted syntax. Each element must be:
1. A simple identifier (e.g., `ElementName`), or
2. An assignment with a constant value (e.g., `ElementName := 123`)

Other statement types are not allowed in enum definitions.
