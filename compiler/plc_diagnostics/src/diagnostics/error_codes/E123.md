# E123: Division by zero error

This error occurs when a literal or constant on the right side of a division operation is zero.

## Examples

### Literal is zero
```st
FUNCTION main
    VAR
        x : DINT;
        divX : DINT;
    END_VAR

    x := 5;
    divX := x / 0;
END_FUNCTION
```

### Constant is zero
```st
VAR_GLOBAL CONSTANT
    ConstantZero: DINT := 0;
END_VAR

FUNCTION main
    VAR
        x : DINT;
        divX : DINT;
    END_VAR

    x := 5;
    divX := x / ConstantZero;
END_FUNCTION
```
