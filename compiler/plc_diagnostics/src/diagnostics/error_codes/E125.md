# E125: Assignment will result in truncation

This assignment will result in truncation and compilation will fail.

In the below example, attempting to assign the REAL `result` value from the `libFunction` to the INT `i1` variable will result in a LLVM error.

```ST
FUNCTION libFunction : INT
    VAR_INPUT
        inVar1 : INT;
        inVar2 : REAL;
    END_VAR
    VAR_OUTPUT
        result : REAL;
    END_VAR
END_FUNCTION

PROGRAM mainProg
    VAR
        i1 : INT;
    END_VAR

    libFunction(
        inVar1 := 0,
        inVar2 := 0.0,
        result => i1
    );
END_PROGRAM
```
