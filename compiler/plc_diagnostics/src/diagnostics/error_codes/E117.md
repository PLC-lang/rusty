# Property with invalid number of PROPERTY_GET and/or PROPERTY_SET blocks

Properties must be non empty and _at most_ contain one block of type PROPERTY_GET or PROPERTY_SET.

Errouneus code example:
```
FUNCTION foo
    PROPERTY bar : DINT
        // Invalid, empty (no PROPERTY_GET or PROPERTY_SET block)
    END_PROPERTY

    PROPERTY baz : DINT
        // Invalid, two PROPERTY_GET blocks
        PROPERTY_GET baz: DINT END_GET
        PROPERTY_GET baz: DINT END_GET
    END_PROPERTY

    PROPERTY qux : DINT
        // Invalid, two PROPERTY_SET blocks
        PROPERTY_SET qux: DINT END_SET
        PROPERTY_SET qux: DINT END_SET
    END_PROPERTY
END_FUNCTION
```
